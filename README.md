# movies-explorer-api

### О чём этот проект:
Backend для дипломного проекта Movies Explorer.

### Технологии:
* Node JS
* Express.js
* MongoDB

### Запуск проекта
**npm run start** — запускает сервер  
**npm run dev** — запускает сервер с hot-reload
